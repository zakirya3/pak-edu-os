Installation Guide

How to Install Pak Edu OS

Follow these four simple steps — from downloading the ISO to running Pak Edu OS on your school computer.

1. Download ISO
›
2. Flash USB
›
3. Boot from USB
›
4. Install OS
1

Download the ISO file

Click the button below to download the Pak Edu OS .iso file from our Google Drive. The file is approximately 2.5 – 3 GB in size — make sure you have enough space and a stable connection (or use a download manager like IDM).

Download ISO from Google Drive
You will need a USB drive of at least 4 GB capacity. Back up any data on it first — it will be erased during flashing.
2

Flash the ISO to a USB drive

Use one of the free tools below to write the ISO to your USB drive. Rufus is recommended for Windows users — it is simple and works perfectly.

Rufus (Windows — recommended) balenaEtcher (Windows/Mac/Linux) Ventoy (multi-ISO USB)

Using Rufus — step by step:

① Open Rufus  →  ② Click Select and choose the downloaded .iso file  →  ③ Select your USB drive under Device  →  ④ Leave partition scheme as MBR  →  ⑤ Click Start  →  ⑥ Wait for it to say READY.

All data on the USB drive will be permanently erased. Double-check you have selected the correct drive before clicking Start.
3

Boot the computer from the USB drive

Insert the USB drive into the school computer and restart it. You need to enter the Boot Menu before Windows loads. Press the correct key for your PC brand the moment the computer starts:

Dell  —  F12
HP  —  F9 or Esc
Lenovo  —  F12
Acer  —  F2 or F12
Asus  —  F8 or Esc
Toshiba  —  F12

In the boot menu, select your USB drive (it may appear as "USB Storage Device" or your USB brand name) and press Enter. The computer will start loading Pak Edu OS from the USB.

If the boot menu key does not work, enter BIOS (usually Del or F2) and change the boot order to put USB first, then save and restart.
4

Install Pak Edu OS

The OS will boot into a live environment — you can try it without installing. When ready, double-click the "Install Pak Edu OS" icon on the desktop.

The installer will guide you through these screens:

① Language — choose Urdu or English  →  ② Keyboard layout — select Urdu (Pakistan) or English (US)  →  ③ Disk setup — for a fresh install choose "Erase disk and install", or "Install alongside" to keep Windows  →  ④ Username & password — create your school account  →  ⑤ Click Install Now and wait 10–20 minutes.

Choosing "Erase disk and install" will delete all files on the computer. Back up any important data before proceeding. For school labs, a clean install is recommended.

When installation is complete, the computer will ask you to remove the USB and restart. Pak Edu OS will load automatically from then on.

First-time terminal update (optional but recommended):
sudo apt update && sudo apt upgrade -y
